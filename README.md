# From LLMs to AI Agents
This repository is for Opencampus course "from LLMs to AI Agents" course.

## Steps to setup this repo
1. Open the terminal in VSCode and create a virtual environment
`python3 -m venv venv`.

2. Switch/Activate to your created virtual environment `venv` using `source venv/bin/activate`.

3. Install the required packages `pip install -r requirements.txt`.

4. You have successfully setup and installed necessarry python packages.
